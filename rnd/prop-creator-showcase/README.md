## Tech Stack

- React + TypeScript
- Three.js via @react-three/fiber + @react-three/drei
- Vite dev server

## Quick Start

```bash
cd rnd/prop-creator-showcase
npm install
npm run dev
# Open http://localhost:5181
```

## Design Guidelines

- Low-poly / flat shading style (matches CrewHub aesthetic)
- Bright, distinct colors per prop
- Subtle animations (rotation, pulsing, floating)
- Dark background theme
