# Zen Mode - Keyboard Shortcuts Reference

Complete reference of all keyboard shortcuts available in Zen Mode.

## Global Shortcuts

| Shortcut | Action | Notes |
|----------|--------|-------|
| `Ctrl+Shift+Z` | Toggle Zen Mode | Works from anywhere in CrewHub |
| `Esc` | Exit Zen Mode | Or close any open modal |
| `Ctrl+K` | Open Command Palette | Works even in input fields |
| `Ctrl+/` or `?` | Show Keyboard Shortcuts | Quick reference overlay |

## Panel Navigation

| Shortcut | Action |
|----------|--------|
| `Tab` | Focus next panel |
| `Shift+Tab` | Focus previous panel |
| `Ctrl+1` - `Ctrl+9` | Focus panel by number |

## Panel Operations

| Shortcut | Action |
|----------|--------|
| `Ctrl+\` | Split panel vertically (side by side) |
| `Ctrl+Shift+\` | Split panel horizontally (stacked) |
| `Ctrl+Shift+W` | Close focused panel |
| `Ctrl+Shift+M` | Maximize/restore focused panel |
| `Ctrl+Shift+Arrow` | Resize focused panel |

## Layout Management

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+L` | Cycle through layout presets |
| `Ctrl+Shift+S` | Save current layout |

## Session Management

| Shortcut | Action |
|----------|--------|
| `Ctrl+N` | New chat with agent picker |
| `Ctrl+Shift+N` | Spawn new agent session |

## Theme

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+T` | Open theme picker |

## Chat (when focused)

| Shortcut | Action |
|----------|--------|
| `Enter` | Send message |
| `Shift+Enter` | New line |
| `Escape` | Cancel streaming / clear input |
| `↑` | Edit last message |
| `Page Up/Down` | Scroll history |

## Command Palette

When the command palette is open:

| Shortcut | Action |
|----------|--------|
| `↑` / `↓` | Navigate commands |
| `Enter` | Execute selected command |
| `Tab` / `Shift+Tab` | Navigate commands |
| `Esc` | Close palette |

## Tips

1. **Command Palette is your friend**: Press `Ctrl+K` for quick access to any action
2. **Fuzzy search works**: Type partial words like "thdr" to find "Theme: Dracula"
3. **Panel numbers**: Panels are numbered left-to-right, top-to-bottom
4. **Safe shortcuts**: `Ctrl+W` (close tab) is intentionally avoided - use `Ctrl+Shift+W` for panels
5. **Layout persistence**: Your layout is automatically saved and restored
